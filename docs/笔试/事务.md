# 事务简介

### 事务的特性 ACID
- 原子性：一个事物内的所有操作，要么都失败，要么都成功  
- 一致性：一个事务执行必须是数据库从一个一致性状态转移到另外一个一致性状态，即事务执行之前和执行之后，数据库必须都处于一致性状态下 
- 隔离性：事务之间不相互干扰 
- 持久性：事务完成之后，数据即永久保存到磁盘
